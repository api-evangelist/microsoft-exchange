# Microsoft Exchange (microsoft-exchange)
A comprehensive API collection for Microsoft Exchange Server and Exchange Online, providing programmatic access to email, calendars, contacts, and other mailbox resources.

**URL:** [Visit APIs.json URL](https://docs.microsoft.com/en-us/exchange/)

## Tags:

 - Email, Calendar, Contacts, Collaboration, Enterprise

Access Exchange Online mailboxes through the Microsoft Graph API, providing modern REST endpoints for email, calendars, and contacts.
