# Microsoft Exchange (microsoft-exchange)
A comprehensive API collection for Microsoft Exchange Server and Exchange Online, providing programmatic access to email, calendars, contacts, and other mailbox resources through Microsoft Graph, EWS, PowerShell, Autodiscover, and the Exchange Online Admin API.

## APIs

9 APIs across Microsoft Graph, EWS, PowerShell, Autodiscover, and Admin surfaces:

| Name | Description |
|------|-------------|
| Microsoft Graph Mail API | REST endpoints for reading, sending, and managing email messages, drafts, attachments, and mail folders. |
| Microsoft Graph Calendar API | Manage calendar events, meetings, attendees, and recurring meetings for Exchange Online users. |
| Microsoft Graph Contacts API | Manage Outlook personal contacts and contact folders for Exchange Online users. |
| Microsoft Graph People API | Retrieve people most relevant to a user based on communication and collaboration patterns. |
| Exchange Web Services (EWS) | Legacy SOAP-based API for Exchange Server providing comprehensive mailbox access. |
| Exchange Online PowerShell API | PowerShell module for managing Exchange Online through REST-based cmdlets. |
| Exchange Autodiscover API | Service that enables clients to automatically configure themselves for Exchange connectivity. |
| Exchange Online Admin API | REST-based administrative API providing cmdlet-style endpoints for Exchange Online management. |
| Microsoft Graph Mailbox Import Export API | Discover, import, and export mailbox content in full fidelity for migration scenarios. |
