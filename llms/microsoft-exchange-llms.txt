# Microsoft Exchange

> A comprehensive API collection for Microsoft Exchange Server and Exchange Online, providing programmatic access to email, calendars, contacts, and other mailbox resources through Microsoft Graph, EWS, PowerShell, Autodiscover, and the Exchange Online Admin API. Authentication is via the Microsoft identity platform (OAuth 2.0 / OpenID Connect). This llms.txt was generated by API Evangelist from the provider catalog (learn.microsoft.com does not publish its own /llms.txt).

## APIs

- [Microsoft Graph Mail API](https://learn.microsoft.com/en-us/graph/api/resources/mail-api-overview): Read, send, and manage email messages, drafts, attachments, and mail folders on Exchange Online. Base URL https://graph.microsoft.com/v1.0
- [Microsoft Graph Calendar API](https://learn.microsoft.com/en-us/graph/api/resources/calendar): Create, update, and delete events, manage attendees, and find meeting times. Base URL https://graph.microsoft.com/v1.0
- [Microsoft Graph Contacts API](https://learn.microsoft.com/en-us/graph/outlook-contacts-concept-overview): Manage Outlook personal contacts and contact folders. Base URL https://graph.microsoft.com/v1.0
- [Microsoft Graph People API](https://learn.microsoft.com/en-us/graph/people-insights-overview): Retrieve people most relevant to a user. Base URL https://graph.microsoft.com/v1.0
- [Exchange Web Services (EWS)](https://learn.microsoft.com/en-us/exchange/client-developer/exchange-web-services/explore-the-ews-managed-api-ews-and-web-services-in-exchange): Legacy SOAP API; deprecation planned for Exchange Online October 2026. Base URL https://outlook.office365.com/EWS/Exchange.asmx
- [Exchange Online PowerShell API](https://learn.microsoft.com/en-us/powershell/exchange/exchange-online-powershell): REST-based management cmdlets via the ExchangeOnlineManagement module.
- [Exchange Autodiscover API](https://learn.microsoft.com/en-us/exchange/client-developer/exchange-web-services/autodiscover-for-exchange): Automatic client configuration (SOAP/POX).
- [Exchange Online Admin API](https://learn.microsoft.com/en-us/exchange/reference/admin-api-overview): REST cmdlet-style admin endpoints. Base URL https://outlook.office365.com/adminapi/v2.0
- [Microsoft Graph Mailbox Import/Export API](https://learn.microsoft.com/en-us/graph/mailbox-import-export-concept-overview): Full-fidelity mailbox content import/export. Base URL https://graph.microsoft.com/beta
